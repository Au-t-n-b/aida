## 核心业务模型

**人货站约束链**：「人等货，货等站」——**站**（机房 / 入场条件）→ **货**（PoD / 设备到位）→ **人**（施工 / 调试队伍）。
以合同交付目标为终点，PoD / 机房为组织粒度。主操作角色：PD（履约统筹）、TD（技术交付），领导走「全局视角」查看与汇报。
