术语表

术语表 · 看不懂上面的词？点开（6 个）+
业务场景 Skill / Skill
把交付里的一个环节（工勘、建模、系统设计…）做成一个能自己跑的 AI 流水线，是本平台的基本单位。
A+B 双层
A 层是一个 SKILL.md（写触发词和端点，给 Claude Code / Cursor 读）；B 层是 Python 代码（skill.py + 各步骤），真正干活。
HITL（人在环中）
流程跑到一半需要你补文件或点确认时会「软中断」停下等你，补完自动接着跑。
SDUI（服务端驱动界面）
后端吐一棵「界面树」、前端照着渲染；新模块不写前端就自动有作业界面。
守门（lint）
一组提交前必跑的检查脚本，自动拦住违反规范的代码——没有守门的规范会退化成 PPT。
Claw · 四层
模块跑在「前端 / Manager / 数据中心 / Claw 执行引擎」四层里；Claw 是真正执行你 Skill 的那一层。
