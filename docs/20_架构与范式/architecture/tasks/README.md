# tasks/ · 实例化任务单

架构师跑 [Workflow A](../00_ARCHITECT_SOP.md#workflow-a-需求分析需求--task) 时，按一个需求生成一份 `TASK_<需求简写>.md`，放在本目录。

- 模板见 [00_ARCHITECT_SOP.md · TASK 模板](../00_ARCHITECT_SOP.md#task-模板复制到-taskstask_名md)。
- 开发者 [D1](../00_DEVELOPER_SOP.md#workflow-d1--领-task-并开发) 领取后，用「标准起手式」`@` 引用本目录对应 TASK。
- 每个 TASK 含：影响速查 / WBS 子任务（修改范围 NEW·MODIFY:行）/ 红线 / 客观可验的验收（AC）。

> 首张试点任务：`TASK_delivery.md`（交付编排模块，P1 阶段生成）。
