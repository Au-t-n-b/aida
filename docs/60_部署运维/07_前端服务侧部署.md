- **唯一数据源**：界面 = `render(state.json)`；实时更新走 **state.json 增量**，*不是* SDUI node patch（04 §5 已判定，以数据中心 README 审计 B3 为准）。
- **取数路径**：经 04 流 ⑥ 调数据中心 API 取 `state.json` + 产物；清单以 `state.outputs` 为准。
- **SDUI 复用**：同一渲染器换 schema 服务多模块（survey/modeling/job）；新模块前端 = 加 schema，不改渲染器。
